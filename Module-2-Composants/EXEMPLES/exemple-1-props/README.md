# Exemple 1 : Props et communication parent-enfant

Projet Vite + Vue 3 pour le Module 2. Démonstration des **props** (données parent → enfant).

## Lancer le projet

```bash
npm install
npm run dev
```

Ouvrez l’URL affichée (souvent `http://localhost:5173`) dans votre navigateur.

## Structure

- `src/App.vue` : composant parent (liste d’utilisateurs, passage des props)
- `src/components/UserCard.vue` : composant enfant (réception des props, affichage)

## Commandes

- `npm run dev` : serveur de développement
- `npm run build` : build de production
- `npm run preview` : prévisualisation du build
