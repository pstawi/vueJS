# Corrections Module 2

Les corrections sont fournies en **fichiers HTML (CDN)** pour une ouverture directe dans le navigateur et une comparaison rapide avec votre solution.

**Pour ce module, la démarche recommandée est le projet avec build (Vite).** Pour voir la même logique sous forme de **Single File Components** et structure Vite, reportez-vous aux dossiers d’exemples :

- `EXEMPLES/exemple-1-props/` → Props, composants
- `EXEMPLES/exemple-2-events/` → Événements personnalisés
- `EXEMPLES/exemple-3-computed-watchers/` → Computed et watchers

Dans chaque exemple : `npm install` puis `npm run dev` pour lancer le projet.
