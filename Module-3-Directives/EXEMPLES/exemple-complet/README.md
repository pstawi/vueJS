# Exemple complet : Directives et événements (Module 3)

Projet Vite + Vue 3 pour le Module 3. Démonstration des **directives** (v-if, v-for, v-model), des **événements** (@click, @submit.prevent, @keyup.esc), des **filtres** et d’un **formulaire** avec validation.

## Lancer le projet

```bash
npm install
npm run dev
```

Ouvrez l’URL affichée (souvent `http://localhost:5173`) dans votre navigateur.

## Contenu

- **Formulaire** : v-model, validation, messages d’alerte
- **Filtres** : v-for sur les catégories, classes dynamiques (:class)
- **Liste produits** : v-for avec :key, v-if / v-else
- **Statistiques** : propriétés computed

## Commandes

- `npm run dev` : serveur de développement
- `npm run build` : build de production
- `npm run preview` : prévisualisation du build
