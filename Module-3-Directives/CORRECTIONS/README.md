# Corrections Module 3

Les corrections sont fournies en **fichiers HTML (CDN)** pour une ouverture directe dans le navigateur et une comparaison rapide avec votre solution.

**Pour ce module, la démarche recommandée est le projet avec build (Vite).** Pour voir la même logique sous forme de **Single File Components** et structure Vite, reportez-vous au dossier d’exemple :

- `EXEMPLES/exemple-complet/` → Gestion de boutique (formulaire, filtres, v-for, v-model)

Dans ce dossier : `npm install` puis `npm run dev` pour lancer le projet.
